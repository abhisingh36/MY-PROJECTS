What is HTML 

 HTML stands for ( Hyper Text Markup Language ) for creating web pages.
 HTML describes the structure of a web page and how it should be displayed.
 HTML consists of a series of elements, which are enclosed in angle brackets (< >).
 Each element has aspecific meaning and tells the browser how to display the content.
